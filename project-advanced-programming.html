App Overview

02
01 / User flowLogin, sign-up, dashboard, and profile.

Users can create an account, enter profile details, and navigate through the app from a personalised dashboard.

02 / Fitness goalsSet targets and track goal achievement.

The app supports calorie, step, weight, and sleep goals with deadline and achievement tracking.

03 / Daily logsRecord workouts and nutrition information.

Workout and meal logging help users store exercise details, calories, macros, and history.

04 / VisualisationTurn fitness data into charts.

Progress views use JavaFX charts to summarise calories, steps, sleep, weight, and goal progress.
